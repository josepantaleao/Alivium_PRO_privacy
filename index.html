Política de Privacidade — Alivium PRO

Extensão Google Chrome · Versão 9.1 · Atualizado em 28/06/2026

1. O que é o Alivium PRO

O Alivium PRO é uma extensão para Google Chrome que automatiza o lançamento de frequência, notas e conteúdo programático nos sistemas SIAP (SEDUC-GO) e Portal do Docente (SENAI-GO). A extensão é operada exclusivamente pelo professor titular da turma, autenticado em sua própria sessão nos portais.

2. Dados tratados

Durante o uso, a extensão lê os seguintes dados para executar as automações:

Dado Origem Finalidade Destino
Nomes de alunos CSV carregado pelo professor Identificar aluno no portal Somente dispositivo
Matrículas (SEDUC / SENAI) CSV carregado pelo professor Localizar aluno no DOM do portal Somente dispositivo
Frequências (P/F por data) CSV carregado pelo professor Preencher checkboxes de frequência Somente dispositivo
Notas por bimestre CSV carregado pelo professor Preencher campos de nota Somente dispositivo
Conteúdo programático Arquivo .docx (FO-178) Preencher calendário de conteúdo Somente dispositivo
Chave de licença Digitada pelo professor no popup Validar licença de uso Servidor de licenças

3. O que NÃO ocorre

Nenhum dado de aluno (nome, matrícula, frequência ou nota) é enviado a qualquer servidor externo. Todo o processamento ocorre localmente no browser do professor. A única comunicação com servidor externo é a validação da chave de licença.

4. Armazenamento local

A extensão utiliza chrome.storage.local para persistir dados temporários entre navegações (necessário porque os portais recarregam a página a cada operação):

Dado armazenadoDuração
CSV de frequência / notas carregadoAté o professor remover ou encerrar a sessão
Estado das automações em andamentoLimpo automaticamente ao reiniciar a extensão
Conteúdo programático (.docx parseado)Até o professor clicar em "Limpar"
Chave e status de licençaPersistente; revalidado a cada 24 horas

Todos os dados temporários com registro de data são removidos automaticamente após 48 horas na inicialização da extensão.

5. Responsabilidades

A extensão automatiza ações que o professor realizaria manualmente nos portais institucionais. O professor é o operador dos dados; SENAI-GO / SEDUC-GO são os controladores. A base legal do tratamento é a execução das atividades pedagógicas pelo docente titular, no exercício de suas funções institucionais.

6. Segurança

Os dados processados nunca saem do dispositivo do professor. A chave de licença é transmitida via HTTPS para o servidor de validação e não é armazenada em texto exposto. A extensão não coleta dados de uso, não registra logs em servidores externos e não exibe publicidade.
